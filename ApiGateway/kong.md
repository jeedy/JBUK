# Kong - API Gateway 
tags: API, gateway

## https://konghq.com/

## 느낌점
spring cloud gateway와 다른점은 직접 소스로 구현을 할 필요가 없는 솔류션으로 보인다.
GUI 어드민 페이지도 제공해 페이지에서 직접 컨트롤하고 지표도 뽑아 볼수있다. 

개발자가 직접 설치하는 것이 아닌 인프라 영역에 가까운 솔류션, 엄블렐라(API umbrella)도 마찬가지이다.
